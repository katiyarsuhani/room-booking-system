# room-booking-system

## Overview

The Room Booking Project is a simple application that allows users to book rooms, check room availability, and manage reservations. It helps reduce manual booking errors and makes the booking process faster and more organized.

## Features

* View available rooms
* Book a room
* Cancel a booking
* Check booking status
